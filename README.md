# cpp_basics
# C++ Basics Programs

This repository contains simple beginner-level C++ programs created during my early learning phase.  
These programs cover the most basic concepts of programming, such as printing output, variables, arithmetic operations, and swapping values.

## Programs Included

### 1. `hello_world.cpp`
A simple program that prints "Hello World!" on the screen.

### 2. `variables.cpp`
Demonstrates basic variable usage in C++ such as int, float, char, string, and bool.

### 3. `calculator.cpp`
A basic calculator that performs addition, subtraction, multiplication, and division using two numbers.

### 4. `swap_numbers.cpp`
Shows how to swap two numbers using a third variable and also without a third variable.
